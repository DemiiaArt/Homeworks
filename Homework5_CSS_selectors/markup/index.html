CSS - Cascadind Style Sheets

3 способи підключення стилів:

  1. інлайново в описі к тегу
  2. прописати в документі html в head
  3. підключити через link сторінку css зі стилями
* {...:...;}
Універсальний (приміняється до всіх елемнтів)
ТЕГ {...:...;}
Теговий (приміняється до конкретного тегу)
.клас {...:...;}
Класовий (приміняється до конкретного класу)
#ІД {...:...;}
Селектор ідентифікатора (приміняється до унікального id)
ТЕГ, ТЕГ {...:...;}
Селектор груп (приміняється до вказаних елемнтів)
а b {...:...;}
Селектор потомка (приміняється до потомка (стоїть другим) вказаного елементу (стоїть на першому місці))
а>b {...:...;}
Дочерній селектор (приміняється до елементу, яки є прямим потомком першого елементу та стоїть одразу за ним)
a+b {...:...;}
Сусідні селектори (приміняється до двох сусідніх елементів. Використовується для більшої компактності написання коду)
a~b {...:...;}
Родственний селектор (приміняється до всіх елементів, які знаходяться після першого елемнту та мають спільний батьківський елемент)
a[атрибут="значення"] {...:...;}
Селектор атрибута (приміняється до вказаних атрибутів)

!important;при записі інлайново перебиває всі інші стилі

Наслідування свойств

Наслідуються відносяться к текстовому змісту

color, font, line-high, word-spacing, white-space, text-align, text-transform, visibility, list-style

Ненаслідуються відносяться к форматуванню блоку

background, border, display, clear, margin, padding, outline, min/max heigh, position, z-index
